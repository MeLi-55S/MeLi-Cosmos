👾 关于

MeLi Cosmos 是一个极简主义个人博客系统，由完整的 Django 驱动，纯 Markdown 写作，Tailwind CSS 渲染。 这里没有臃肿的第三方 CMS 插件，没有复杂的前端框架，只有纯粹的文字和代码。

🛠️ 技术栈

  • · Django 6.x（Python 3.14+）— 后端框架
  • · Tailwind CSS v4 — CDN 原子化 CSS
  • · Python-Markdown + Pygments — Markdown 与代码高亮
  • · SQLite / PostgreSQL — 数据库
  • · Nginx + Gunicorn + Cloudflare — 生产部署

📡 理念

最小化界面，最大化控制。所有内容以纯 Markdown 格式摄入，与 Obsidian 通过 UUID 同步。 严格分离后端业务逻辑与前端样式。零模板依赖。
